mdbox控制相关参数：

1、控制器IP：fn020~fn023
初始为192.168.15.201和192.168.15.202

2、控制器端口号：fn024,fn025

3、上位机UDP端口号：fn034,fn035

4、动感平台初始定位参数
	fn00D,电机定位原点到超程点距离（注意X,Y,Z复位时是顺时针转，U,V,W是逆时针转）
	当fn00D设置为-1时，可独立设置各轴定位参数，具体为fn0B1~fn0B6

（复位值：304.583）

具体参考MDBOX使用手册。以上几个主要参数，均可在使用手册中查询到

PS：建议在初步调试时，将上平台卸下。只保留电机连杆（20cm长的那个），复位时电机会自动卡到底，然后反向走一定距离（该距离就是fn0B1~fn0B6所能调节的）。同时，测试过程中，若出现卡死的情况，控制器为了保护电机会跳转至报错状态并紧急停机，因此电机还是较为安全的。所以，调试者要注意安全~
